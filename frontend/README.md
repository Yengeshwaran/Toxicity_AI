# 🧬 CodeCure — Frontend

> React-based frontend for the **ToxiGuard AI** drug toxicity prediction system, built with **Vite**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**.

This frontend provides a clean, modern interface for submitting SMILES strings and visualizing toxicity predictions, molecule structures, SHAP feature analysis, AI-generated explanations, and downloadable PDF reports — all powered by the ToxiGuard AI backend API.

## ⚙️ Setup & Installation

### Prerequisites

- **Node.js 18+**
- **Backend server running** at `http://127.0.0.1:8000` (see backend README)

### 1. Install dependencies

```bash
cd frontend
npm install
```

### 2. Start the development server

```bash
npm run dev
```

The app opens at **http://localhost:5173** (default Vite port).

> **Important:** Make sure the backend server is running on port 8000 before using the frontend.

---

## 🔗 Backend Connection

The frontend connects to the backend API at:

```
http://127.0.0.1:8000
```

This is configured in `src/pages/Index.tsx`:

```typescript
const API_BASE_URL = "http://127.0.0.1:8000";
```

### API Endpoints Used

| Endpoint | Method | Purpose |
|---|---|---|
| `/predict` | POST | Send SMILES string, receive toxicity prediction |
| `/report/download/{filename}` | GET | Download generated PDF report |

## 🖥️ Features & UI Sections

### 1. SMILES Input
- Text input with Enter key support
- Quick-try buttons for common molecules (Ethanol, Benzene, Aspirin, Acetaminophen, Caffeine)
- Input validation with error display

### 2. Prediction Result Card
- Displays **Toxic** (red) or **Non-toxic** (green) prediction
- Shows model **confidence score** as a percentage
- Color-coded badge and icon based on toxicity

### 3. Molecule Structure
- Renders the 2D molecular structure from the backend's base64-encoded PNG
- Displayed on a white background card for clarity

### 4. SHAP Feature Analysis
- Lists the top contributing molecular features identified by SHAP
- Ranked with numbered badges (#1, #2, etc.)
- Includes a brief explanation of SHAP methodology

### 5. AI-Powered Toxicity Analysis
- Displays the detailed explanation generated by Google Gemini AI
- Labeled with a "Gemini" badge
- Rendered inside a styled container with `whitespace-pre-line` for proper paragraph formatting

### 6. PDF Report Download
- Full-width download button at the bottom of results
- Opens the backend's PDF report in a new tab
- Report contains all prediction data + AI analysis

## 🚀 How to Use

1. **Start the backend** (from the `backend/` folder):
   ```bash
   .\venv\Scripts\activate
   uvicorn src.api:app --reload
   ```

2. **Start the frontend** (from the `frontend/` folder):
   ```bash
   npm run dev
   ```

3. Open **http://localhost:5173** in your browser

4. Enter a SMILES string (or click one of the quick-try buttons)

5. Click **Predict** — the results will appear with:
   - Toxicity prediction & confidence
   - 2D molecule image
   - SHAP top features
   - AI-generated analysis
   - Download button for the PDF report
